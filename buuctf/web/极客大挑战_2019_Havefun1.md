## 解法

devToolを開くとコメントに以下の内容が書かれている．

```HTML
<!--
        $cat=$_GET['cat'];
        echo $cat;
        if($cat=='dog'){
            echo 'Syc{cat_cat_cat_cat}';
        }
        -->
```

即ち，クエリに`?cat=dog`を設定することでフラグが手に入る．
